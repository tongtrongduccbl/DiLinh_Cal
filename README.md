# TTD Ver 1.1 - Medical Calculator Suite

Bộ công cụ tính toán y khoa chuyên nghiệp dành cho bác sĩ, được phát triển bởi **BS. Tống Trọng Đức**.

## 🌟 Tính năng chính

Ứng dụng cung cấp các công cụ tính toán lâm sàng quan trọng:
- **Tim mạch**: CHA₂DS₂-VA, ASCVD, HAS-BLED, LDL-C, SCORE2, PREVENT, Z-Score HF.
- **Thận học**: Creatinine Clearance (Cockcroft-Gault), eGFR (CKD-EPI).
- **Tiêu hóa**: Child-Pugh Score.
- **Hồi sức & Cấp cứu**: Glasgow (GCS), CURB-65, SOFA Score.
- **Mạch máu**: ARC-HBR (High Bleeding Risk).

## 🚀 Công nghệ sử dụng

- **Frontend**: React 18, TypeScript, Vite.
- **Styling**: Tailwind CSS.
- **UI Components**: Radix UI, Lucide Icons.
- **Animation**: Framer Motion.

## 🛠 Cài đặt và Chạy thử

Để chạy ứng dụng trên máy cục bộ, hãy làm theo các bước sau:

1. **Clone repository**:
   ```bash
   git clone <your-repo-url>
   cd <repo-name>
   ```

2. **Cài đặt dependencies**:
   ```bash
   npm install
   ```

3. **Chạy môi trường phát triển**:
   ```bash
   npm run dev
   ```

4. **Build sản phẩm**:
   ```bash
   npm run build
   ```
